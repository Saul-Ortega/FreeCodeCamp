Introduction

TypeScript is a free open-source high-level programming language developed by Microsoft which adds static typing with optional type annotations to JavaScript. Its designed to be used in the development of large applications and with the purpose of make them more secure. You could find them with the extensions: .ts, .tsx, .mts, .cts.

Hello World!

The tradition of using the phrase “Hello, world!” was influenced by an example in the book of “The C Programming Language” and is also used to make sure the computer language was correctly installed.

let message: string = “Hello, world!”;
console.log(message);
Variables

Var: Is only available inside the block where they are defined.
Let: Is available throughout the function in which they are declared.
Const: All declarations other than those you plan to modify should use this declaration.

Data Types

String: Represents string values like “Hello World!”.
Number: JavaScript does not have a special value like integers or float, the are only numbers.
Boolean: Is for the 2 values true or false.
Any: Special type that you can use when you don’t want a particular value and every variable that is initialized with null or undefined, will have this type.
