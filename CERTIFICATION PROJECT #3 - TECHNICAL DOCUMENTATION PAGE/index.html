Introduction

TypeScript is a free open-source high-level programming language developed by Microsoft which adds static typing with optional type annotations to JavaScript. Its designed to be used in the development of large applications and with the purpose of make them more secure. You could find them with the extensions: .ts, .tsx, .mts, .cts.
